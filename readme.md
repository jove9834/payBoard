## 属性

* #### users/mobile

---

获取用户手机号

##### 是否需要权限验证

---

是

##### 请求参数

---

| | 必选 | 类型 | 说明 |
|:-------------:|:-------------|:-------------|
| userId | true | int | 用户ID |

##### 请求方法

---

GET

##### 调用样例

---


` ``
users/mobile?userId=214
` ``
